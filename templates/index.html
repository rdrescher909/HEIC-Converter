Convert HEIC images to png

This is a simple tool to convert HEIC images to png.


There is no limit to the number of images that you can submit, however the maximum size for all of the images after processing is 32MB. Generally this means that you can upload about 3 iPhone 12 Pro Max HEIC images at a time.
